# Game UI Design - 蒲牢出游游戏UI设计

这是一个游戏UI设计和交互演示项目，使用 HTML5、CSS3 和 JavaScript 实现。

## 项目特性

- 🎮 交互式游戏界面设计
- 📱 响应式移动设备支持
- ✨ 流畅的动画效果
- 🎯 关卡选择系统
- 📖 日记和图鉴功能
- 🏆 游戏胜利/失败界面

## 快速开始

### 安装依赖

```bash
npm install
```

### 运行开发服务器

```bash
npm start
```

服务器将在 `http://localhost:3000` 启动

### 检查代码

```bash
npm run check
```

## 项目结构

```
.
├── index.html              # 主页面
├── mobile-preview.html     # 移动设备预览页面
├── app.js                  # 主应用逻辑
├── server.mjs              # Node.js 服务器
├── styles.css              # 样式文件
├── package.json            # 项目配置
├── vercel.json             # Vercel 部署配置
└── assets/                 # 资源文件夹
    ├── effects/            # 特效和图标
    └── page-*.jpg          # 游戏界面截图
```

## 技术栈

- HTML5
- CSS3
- JavaScript (Vanilla)
- Node.js
